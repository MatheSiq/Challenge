Tecnologias Utilizadas na construção do projeto.

O Site foi desenvolvido utilizando a ferramentas de HTML, CSS e Javascript. O HTML foi utilizado em todas as páginas do site, pois nele está todo o conteúdo, como o menu da parte superior, os vídeos da página "Home", nos formulários presentes nas páginas de "Abertura de Chamado" e "Chatbot". Assim como o HTML, o CSS está presente em todas as páginas para estilizar o HTML, porém os arquivos CSS estão praticamente crûs, pois foi utilizado um framework para a construção do site, o Bootstrap. Com o Bootstrap a criação do site foi extremamente acelerada com a facilidade de construir o HTML e aplicar as classes corretas, assim o projeto já possui uma estilização básica e passa a ser responsivo. Para a paleta de cores do site foram utilizadas ferramentas externas, assim como foi utilizado IAs generativas para a correção e enriquecimento de alguns textos da página.
